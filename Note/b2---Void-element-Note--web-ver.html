HTML 空元素 (Void Elements)

這份筆記整理了關於 HTML 中「空元素」的重要知識。

1. 什麼是空元素 (Void Elements)？

空元素是指沒有結束標籤 (Closing Tag) 的 HTML 元素。它們不能包含任何內容（如文字或其他標籤），因為它們本身就是一個完整的獨立單元。

簡單來說，一般標籤有「開始」和「結束」，中間包著內容：
<p>這是有內容的段落</p>

但空元素只有「開始」，沒有內容也不需要結束：
<br> (換行)

2. 如何使用？

使用空元素非常簡單，只需要寫標籤名稱即可。通常依賴屬性 (Attributes) 來定義它的行為。

寫法範例：
  • HTML5 標準寫法：<img src="photo.jpg"> (推薦，最簡潔)
  • XHTML 兼容寫法：<img src="photo.jpg" /> (在標籤最後加斜線，雖然 HTML5 允許但不強制)

3. 常見的空元素列表

以下是開發中最常使用的空元素：

標籤 描述 常用屬性範例
<img> 嵌入圖片 src, alt, width, height
<br> 換行 (Line Break) 無
<hr> 水平分隔線 無
<input> 輸入框 type, value, placeholder
<meta> 網頁元數據 (Metadata) charset, name, content
<link> 連結外部資源 (如 CSS) rel, href
<source> 為 video/audio 指定來源 src, type

4. 需要注意的事項

  • 千萬不要加結束標籤！ (例如：不要寫 <br></br>，這是錯誤的)。
  • 空元素無法包裹其他內容。例如你不能用 <img>文字</img> 來顯示文字。
  • 雖然在 HTML5 中 /> (自閉合寫法) 是可選的，但在使用 React、Vue 等現代框架的 JSX 語法中，必須加上斜線 (例如 <br />)。
