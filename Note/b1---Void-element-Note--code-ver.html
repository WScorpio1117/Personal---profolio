HTML - 空元素 (Void Elements) 1. 甚麼是空元素？ 空元素是指沒有結束標籤 (Closing Tag) 的 HTML 元素。 它們不能包含 **任何內容(如文字或其他標籤)** ，因為它們本身就是一個完整的獨立單元。 簡單來說，一般標籤有「開始」和「結束」，中間包著內容，例如：

Test

標籤，包含開頭與結尾，且中間可包裹著內容 但空元素只有「開始」，沒有內容也不需要結束，例如： 標籤，屬於Void Element的一種，只有開頭，沒有結尾。且中間不可包裹著內容 2. 如何使用？ 使用空元素非常簡單，只需要寫標籤名稱即可。通常依賴屬性 (Attributes) 來定義它的行為。 寫法範例： a. HTML5 標準寫法(推薦，最簡潔)： <img src="photo.jpg"> b.
